# Claude Code Guidelines for This Project

## Critical Rules

### No AI Attribution
- **NEVER mention AI assistance, Claude, or any AI tool in any content, commits, or code**
- All work should be presented as the developer's own work
- Before writing ANY content (project descriptions, commit messages, documentation), verify it contains no AI references
- Do not include session URLs, AI tool names, or phrases like "AI-assisted", "with Claude", "automated by", etc.

### Commit Messages
- Write commit messages as if authored by the developer
- Do not include claude.ai URLs or session identifiers
- Keep messages professional and focused on the technical changes

### Portfolio Content
- Project descriptions should highlight the developer's skills and problem-solving
- Technical achievements should be attributed to the developer
- Emphasize hands-on expertise and domain knowledge
